# WebSocket-Android说明
本Demo实现的功能：

1.与websocket建立长连接

2.与websocket进行即时通讯

3.Service和Activity之间通讯和UI更新

4.心跳检测和重连（保证websocket连接稳定性）

5.服务（Service）保活.通过测试可在app手动退出的情况下，服务任运行
